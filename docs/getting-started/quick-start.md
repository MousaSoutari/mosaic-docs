# Quick Start

## Product Development (bmad-bmm)

~~~bash
/plugin install bmad-core@mosaic-plugins
/plugin install bmad-bmm@mosaic-plugins
/bmad-setup
~~~

Then:
~~~bash
/bmad-agent-bmm-pm              # Launch PM agent for PRD creation
/bmad-bmm-create-prd            # Or jump straight to PRD workflow
/bmad-bmm-create-architecture   # Then architecture
/bmad-bmm-create-epics-and-stories  # Break into stories
/bmad-bmm-dev-story             # Implement stories
~~~

## Game Development (bmad-gds)

~~~bash
/plugin install bmad-core@mosaic-plugins
/plugin install bmad-gds@mosaic-plugins
/bmad-setup
~~~

Then:
~~~bash
/bmad-agent-gds-game-designer   # Launch game designer agent
/bmad-gds-create-game-brief     # Define your game vision
/bmad-gds-create-gdd            # Full game design document
/bmad-gds-game-architecture     # Technical architecture
/bmad-gds-dev-story             # Implement stories
~~~

## Instructor Mode (bmad-edu)

~~~bash
/plugin install bmad-core@mosaic-plugins
/plugin install bmad-edu@mosaic-plugins
/bmad-setup
~~~

Then follow the 4-phase pipeline:
~~~bash
/edu-define-course              # Phase A: Define course
/edu-build-project              # Phase B: Build with captures
/edu-generate-course            # Phase C: Generate all materials
/edu-create-tutorial-project    # Phase D: IET Essentials Pathway
~~~
